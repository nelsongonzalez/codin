# Picture element

> this element is not polyfilled for IE, so sources will not change.

## example
```html
<cwc-picture cover>
 <source media="(min-width: 400px)" srcset="./high-res.jpg">
 <source srcset="./low-res.jpg">
</cwc-picture>
````
